# mask-validator

### Requirements

- python 3.7
- PyQt5 (5.15.4)
- opencv-python (4.5.2.52)

### Getting started

1. install package
``` bash
$ cd %project_dir%
$ bash setup.sh
```

2. execute by icon on the desktop

3. press `open dir...` to process
